  1. 三位组成的数列有等差数列 : 6个球一共有1,107,568种组合，这是小学就会的；6个球中有任意四个至少一个成等差数列的很少，占总数5%；但是任意三个至少一个成等差数列的很多，占总数63%
  2. 新一期的号码分割更有可能出现在间隔中 : 如果按照从小到大排序，(1,2,3,4,5,6)是第一位，(28,29,30,31,32,33)是最后一位。目前出现的所有1377期中，相邻两个直接的间隔，就是位置之差。有这样的特点：最大间隔4586，最小间隔1。其中， 339个间隔超过1000，而且有碎片化的趋势，即新一期的号码分割更有可能出现在这些间隔中
  3. Combination(6,3)频率是（0，1，2）：6个号码的任意三位组成的数列，Combination(6,3)共20个。如果按照重复率统计的话。新一期6个号码任意三位组成的数列出现的重复率不超过2，即这20个三位数列的频率是（0，1，2），即在以往所有出现过的6个号码任意三位组成的数列中，新一期6个号码任意三位组成的数列将出现的频率是：0次（没出现过）、1次（出现过1次）、2次（出现过2次，这类一般只有一个）
  4. 如果在这些超过1000的间隔里面，取出某个区间，重新筛选出无4阶等差、有3阶段等差、3阶段等差仅含（0，1，2），最后还有重复抽样的概率约束

交大10年，不能没有晨光BT，如今已经叫PT了。以前ubuntu外网不需要设置，直接安装gw6c即可，记得去年突然失效，后来在linuxmint下可以直接运行。一度以为ubuntu out了，百思不得其解。今日偶得一帖，迁移至gogoc，问题解决，具体如下： ubuntu 13.10下，新建6to4隧道

目前gw6c已经变成gogoc，需要编辑/etc/gogoc/gogoc.config

vi /etc/gogoc/gogoc.conf

将server=anonymous.freenet6.net

设置为：

server=tb.ipv6.apol.com.tw

sudo gogoc

ps -ef | grep gogoc

ifconfig

可以看到tun，说明成功了，CGPT
